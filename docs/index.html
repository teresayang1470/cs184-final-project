Preliminary Results

Using the Taichi programming language, which is discussed further in the “Current Progress” section below, we have been able to render rigid, elastic, and viscoelastic materials; such as various shaped rigid and elastic bodies and specific materials such as water and snow.

The following renders were made using the Minimum Least Squares Material Point Method, which improves the way in which traditional MPM does the discretisation that projects the continuous quantities to discrete particles and nodes. We expanded upon and reworked a basic engine to produce these results. We gained a conceptual basis with the rudimentary given demos to gain an understanding of how the materials are simulated and adjusted the parameters and logic in the engine accordingly to simulate the desirable material effects we wanted.


Current Progress

So far, we have been focusing on exploring the Taichi programming language as a means of adding support for more materials in the MLS-MPM engine. Taichi is a Python-based programming language optimized for computer graphics, using Python as a framework. We will continue to modify and expand the current MLS-MPM engine to be more robust and generic.

Our original plan was to convert a traditional MPM engine to a MLS-MPM engine by implementing MLS discretization, but we are pivoting and are planning on taking a MLS-MPM engine and improving upon that by expanding the engine to support more materials types and shapes. We plan to add expansion and support for a diverse set of materials that we can manage ranging from expanding support for material shapes, realistic hair simulations, lava, sand, and other viscoelastic materials.

In addition to the preliminary renders displayed above, we have learned about the physics concepts involved. For example, from reading code and literature, we discovered the effects of various constants in terms of the physical properties. Young’s modulus ($E$) measures tensile elasticity and determines how susceptible a body is to deformation along a given axis when opposing forces are applied along the axis. The Poisson ratio ($\nu$) measures the deformation of an object perpendicular to an applied force. Softer materials generally have higher values of $\nu$. Lamé’s second parameter ($\mu$) describes a similar measure of tendency to shear (along axis coplanar to the applied force) stress. Finally, the first Lamé parameter ($\lambda$) is used to compute the Cauchy stress tensor ($\sigma$), which gives the stress state for a uniformly stressed body In $\texttt{mpm_solver.py}$, these parameters are varied across different materials to simulate various hard and soft bodies; since the constants affect the stresses and deformation, they are also used compute momentum changes and therefore affect the overall simulation.

$$\tag{1} \lambda = \frac{E \nu}{(1 + \nu)(1 - 2 \nu)}$$ $$\tag{2} \mu = \frac{E}{2(1 + \nu)}$$ $$\tag{3} \sigma = 2 \mu \epsilon + \lambda \text{tr}(\epsilon)I$$

Here, $\text{tr}$ is the trace function, and $\epsilon$ is the strain tensor.

Obtaining an understanding of the physics and math behind MLS-MPM has been helping us expand our engine making it more versatile and has helped us make our materials look physically realistic and accurate. We are hoping to now use this knowledge to move onto a diverse set of hair simulations or viscoelastic fluid simulations.
